# 🐾 PETiSoin — Votre assistant personnel pour animaux de compagnie 🐶🐱

---

## 📖 Présentation

**PETiSoin** est l'application que vous allez créer ensemble au fil de ce cours.  
Conçue pour les amoureux des animaux, PETiSoin est un assistant personnel dédié à la gestion et au soin des compagnons à quatre pattes.  

---

## 🚀 Fonctionnalités principales

- 📋 **Carnet de santé numérique** pour chaque animal  
- ⏰ **Rappels automatisés** pour vaccins, traitements et rendez-vous vétérinaires  
- 🍽️ Suivi des **habitudes alimentaires**  
- 🏃‍♂️ Enregistrement des **exercices quotidiens**  
- 📝 Prise de **notes personnalisées** pour tout ce qui concerne le bien-être de votre compagnon  
- 📶 **100% hors-ligne** : accès garanti à toutes les données, même sans connexion internet  

---

## 🎯 Objectif du projet

Permettre à chaque utilisateur de garder toutes les informations vitales sur ses animaux accessibles, simples à consulter et à mettre à jour, où qu'il soit.  

---

## 🛠️ Technologies utilisées

- React Native / Flutter (ou autre selon choix)  
- Base de données locale (SQLite, Realm...)  
- Notifications locales  
- Gestion hors-ligne avancée
